October 21, 2017 · TCP/IP 初学者心态

LEARN TCP/IP PROTOCOL

晚上花了一个多小时的时间看完了《TCP/IP协议》第一卷第一章概述，学到了很多知识。加之用MindNode软件边看书边画脑图，效果
非常好，还是很有信心学习好TCP/IP协议的。任何工程行业都有自己的不变的知识，需要花时间长期积累，年轻的时候可以为了赚快钱，可以不学习这些东西，但终究会遇到那个所谓的瓶颈。而突破这个瓶颈的唯一方法就是，扎到更加深入的去研究，去学习，除非转行。

昨天终于第二把过了科目三，离开车只剩下一步之遥，考驾照这一年，简直是我人生中的一个魔咒，不是说有多难，而是细节点多，随机性大，学得好不一定考得好，大大小小的失误总是会犯。其实这也折射出我的临场心理素质不是很好，至少没我媳妇那么好。

十一刚过三十岁生日，我就开始rework自己的人生，写了一个大的MindNode，把能够写的都写上，这周写完之后，我要花时间重塑我的大脑，去找到适合我人生的方法。

早起，作为第一步。
